Animation for the math explainer video "What's the probability that a broken stick can form a polygon?". 

This is a script written to be converted to an animation by MANIM (Mathematical Animation Engine), based on Python. I developed this to visually explain the solution to a probability problem. This was a candidate for the 2022 Summer of Math Exposition.
